Download (Far2 x86/x64)

About

When you get broken text, while pasting from Windows clipboard to the Far 2, you may use this plugin, which fix clipboard contents.

Usually, you'll get broken text, while copying national characters from old (non Unicode) programs, when English keyboard layout is active.

При копировании в буфер обмена русского текста из старых (не юникодных) программ и последующей вставки в FAR2 - вставляются "крякозябры". Сей плагин исправляет проблему.

Installation

Unpack appropriate version to the Plugins\ClipFixer subfolder and install reg file.

Установить в Plugins и запустить reg файл.
