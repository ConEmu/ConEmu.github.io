Download (Far3 x86/x64)

About

This plugin allows using of Far2 plugin under Far3.

1. Unpack distribution contents to %FARHOME% directory.

2. Copy Loader.dll (Loader64.dll using Far3 x64) to the directory, containing Far2 plugin binary (i.e. where colorer.dll located).

3. (optionally) Rename Far2 plugin binary (i.e. colorer.dll -> colorer.dl).

4. (optionally) Rename Loader.dll to more obvious name (i.e. colorer3.dll).

5. (optionally) Create wrapper ini file. It must match exactly with wrapper file name, renamed on step 3 (i.e. colorer3.ini). If you don`t create ini-file, wrapper will create it for you with unique GUIDs. !!!WARNING!!! When you create ini-files manually, you must create unique GUIDs in each ini wrapper file.

Sample wrapper ini files are included in distribution.
